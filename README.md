### Installation

1. Clone the repo
```sh
git clone https://github.com/Thibault-collard/thibault.collard-bovy.git
```
2. Instal NPM packages
```sh
npm install
```
4. Build and launch the server
```sh
npm run start
```
To replicate this project, you don't need any type of credentials or API Key
